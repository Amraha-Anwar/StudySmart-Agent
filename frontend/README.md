# StudySmart Agent - Frontend

This directory contains the frontend for the StudySmart Agent application. It is a Next.js application that provides the user interface for uploading PDFs, viewing summaries, and taking quizzes.

## Getting Started

### Prerequisites

- Node.js and npm (or yarn)

### Installation and Running

1.  Install the dependencies:
    ```bash
    npm install
    ```
2.  Start the Next.js development server:
    ```bash
    npm run dev
    ```
    The application will be running at `http://localhost:3000`.

## Features

- **PDF Upload:** A drag-and-drop interface to upload PDF files.
- **Summary Display:** A clean and modern UI to display the summarized notes.
- **Interactive Quiz:** An interactive quiz interface that supports multiple-choice, true/false, and short answer questions.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Shadcn UI
- Framer Motion
